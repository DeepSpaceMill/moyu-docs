在本章节中，你将快速了解如何使用官方提供的标准框架来构建一个简单的视觉小说。我们将从安装引擎开始，逐步引导你完成一个基本的视觉小说项目。

## 必要准备

在开始之前，请确保你已经安装了以下工具：

- [Node.js](https://nodejs.org/)（建议使用最新的 LTS 版本）
- [Git](https://git-scm.com/)（用于克隆项目模板，非必需）

## 安装标准框架

:::tip[什么是框架？]
框架是与引擎本体配套的一个前端工程项目，其提供给开发者标题界面、舞台、存读档、音乐鉴赏等界面组件，内置各种可直接调用的命令实现，并且包含了一个示例剧本。你可以在此基础上进行修改和扩展，来创建属于你自己的视觉小说。
:::

我们提供了一个官方的标准框架，包含了基本的文件结构和示例代码。你可以通过以下命令克隆这个框架：

```bash
git clone https://github.com/DeepSpaceMill/framework.git my-vn-project
```

如果你不想使用 Git，也可以直接从 GitHub 上下载 ZIP 文件并解压到你硬盘上的任意位置。

然后，进入项目目录并安装依赖：

```bash
cd my-vn-project
npm install
```

## 安装和更新引擎本体

当前我们只下载并安装了框架的代码，但引擎本体是一个独立的程序，需要单独安装。
你可以通过以下命令来安装引擎：

```bash
npm run engine:update
```

这一命令用于从远端服务器获取最新的引擎版本，并将其安装到你的项目中。请确保在运行此命令之前已经连接到互联网。

## 开发和预览

安装完成后，你可以通过以下命令启动开发服务器：

```bash
npm run dev
```

之后，开启一个新的控制台窗口：

如果你想要使用本地程序来预览你的视觉小说，可以运行以下命令：

```bash
npm run engine:native
```

如果你更喜欢在浏览器中预览，可以运行：

```bash
npm run engine:web
```

二者均支持热更新，这意味着你在编辑代码时，预览窗口会自动无刷新更新以显示最新的更改。
